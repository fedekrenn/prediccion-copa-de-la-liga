# Comportamiento del agente

- Las tareas simples y bien definidas se ejecutan directamente.
- No asumir requisitos implicitos cuando haya ambiguedad material.
- Si una peticion sigue sin estar clara despues de revisar el repo, hacer una pregunta concreta antes de ejecutar.
- En cambios complejos, confirmar entendimiento solo cuando el alcance o las restricciones no puedan inferirse con seguridad.
