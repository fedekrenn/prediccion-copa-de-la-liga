# Arquitectura y organizacion

- TypeScript es obligatorio.
- Preferir ESM y sintaxis moderna del navegador.
- Componentes pequenos, con una sola responsabilidad.
- Preferir composicion frente a configuraciones complejas.
- Evitar abstracciones prematuras.
- El codigo compartido debe vivir en carpetas claras como `components`, `layouts`, `lib` o `utils`.
